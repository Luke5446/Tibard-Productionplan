Tibard · Production Capacity Planner
v3 · queue & machine load

Work orders

Live works orders from the buffer app — each SKU on a WO becomes its own routed job. They run first-come-first-served in the order below; drag the grip to push a job up the queue, or set a sewing machine.

Load board

Capacity used per line, per working day, summed from each machine's hours. Turning a machine off lowers this automatically.

quietbusytightfull· % of that day's capacity

Pipeline

Where each order sits today and its forecast finish against the due date.

Cutting

Hours and the queue of work orders per day. Drag a WO chip onto another to change the cutting order.

Sewing

Machines grouped by line, each showing the orders queued on it per day. Drag chips to reorder. Hats run on apron or flex at 85/day.

Finishing

One team — set each person's hours individually for holidays. Stud 1 and Stud 2 each take a person off the tables when running.

Master times

The standard throughput each station runs at. These drive every forecast — change one and the whole plan recalculates. Friday and holidays scale automatically from the hours you set per machine.
